- Real-time status indicator

### 2. Text-to-Speech (Voice Output)

- Click "Speak" button to replay last message

- Adjustable speech rate and pitch

### 3. Speech-to-Speech

## How to Use New Features

### Voice Input
1. Click "Mic" button
2. Speak your message
3. Click "Stop" when done
4. Message appears in input field
5. Click "Send" to submit

### Voice Output
1. Bot messages auto-speak
2. Click "Mute" to stop
3. Click "Speak" to replay
4. Adjust volume in browser

### File Upload
1. Click "Files" button
2. Choose "Upload Image" or "Upload Document"
3. Select file from device
4. Wait for upload confirmation
5. File ID displayed in chat

### Dark Mode
1. Click "Dark" button in header
2. Interface switches to dark theme
3. Preference saved automatically
4. Click "Light" to switch back

### Font Size
1. Use Small/Medium/Large buttons
2. Text size changes immediately
3. Preference saved
4. Applies to all messages

### Chat History
1. Click "Save" to save current chat
2. Click "History" to view saved chats
3. Click on a chat to load it
4. Click "Delete" to remove

### Export Chat
1. Click "Export" button
2. Text file downloads automatically
3. Open in any text editor
4. Contains full conversation
